Devo is an open source coding agent for developers who want a local,
terminal-first workflow and the ability to connect to their own model provider.

The docs are organized for new users, Codex CLI and Claude Code users,
provider integrators, and contributors who need to understand the runtime before
changing it.

## Documentation Areas

- Product docs: Get Started and Core Concepts explain what Devo is, how to
  install it, how models/providers work, and which features exist.
- User guide: Using Devo explains the TUI workflows you use day to day.
- Developer reference: Configuration, Reference, and Contributors document
  provider wiring, commands, config fields, runtime flow, and contribution
  paths.
