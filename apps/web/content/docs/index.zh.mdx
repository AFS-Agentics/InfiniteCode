Devo 是一个开源编程 Agent。它跑在本地终端里，可以接 DeepSeek、Anthropic、OpenAI-compatible gateway，或你自己的模型服务。

先把 Devo 跑起来。然后再看 TUI、模型配置、上下文、MCP、skills、hooks 和贡献入口。

## 文档结构

- Get Started 和 Core Concepts：安装、模型、上下文和扩展机制。
- Using Devo：TUI 里的日常操作。
- Configuration、Reference 和 Contributors：配置字段、provider 接入和运行时结构。
