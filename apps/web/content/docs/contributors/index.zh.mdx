## 工作规则

按 crate 边界改：

| Layer | Owns |
| --- | --- |
| `devo-cli` | 进程入口和命令行解析。 |
| `devo-tui` | 终端 UI 状态、composer 行为、overlay 和渲染。 |
| `devo-server` | Session runtime、持久化、事件协议、goals、subagents 和命令执行。 |
| `devo-core` | Agent loop、tools、context、compaction、permissions、hooks 和 instruction discovery。 |
| `devo-provider` | Provider request 和 streaming adapter。 |
| `devo-config` | 序列化配置 schema 和 effective config loading。 |
| `devo-protocol` | 跨 crate 边界共享的 DTO。 |

功能跨越 TUI/server 边界时，先更新 protocol types，再更新 server handling、worker mapping 和 TUI rendering。
