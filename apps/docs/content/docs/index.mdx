InfiniteCode is an open source, model-neutral coding agent for
developers who want a local, terminal-first workflow. You can run it
as a single binary, talk to it from the Desktop app, or wire it into
the editor of your choice.

This site carries the product documentation. Navigation lives in the
sidebar.

## What's new in v0.1.31

The v0.1.31 release ships four new capability surfaces:

- **[Self-Review workflow](/docs/features/self-review)** — Best-of-N
  proposal selection plus multi-perspective audit, before any code
  reaches your repository.
- **[Desktop voice input](/docs/features/desktop-voice)** —
  Push-to-talk dictation in the Desktop composer.
- **[Native web search](/docs/features/desktop-search)** — Provider-routed
  web search with inline citation chips.
- **[Agent behavior config](/docs/features/agent-behavior)** — CLI
  flags and env vars for `self_verify`, `suggest_followups`,
  `compact_strategy`, and `explore_solutions`.

## About InfiniteCode

InfiniteCode is structured for users who want auditable, low-drama
agentic workflows. The CLI and the Desktop talk to the same Rust
runtime, the orchestrator is small, the prompts are versioned, and
the providers are explicit.
